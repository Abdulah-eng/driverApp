# ============================================
# SETUP INSTRUCTIONS:
# ============================================
# 1. Copy this file and rename it to .env (remove .example.txt)
# 
# 2. SUPABASE SETUP:
#    - Go to https://app.supabase.com
#    - Select your project (or create a new one)
#    - Go to Settings → API
#    - Copy the "Project URL" → SUPABASE_URL
#    - Copy the "anon public" key → SUPABASE_ANON_KEY
#
# 3. GOOGLE MAPS SETUP (Android):
#    - Go to https://console.cloud.google.com/google/maps-apis
#    - Create a new project or select existing
#    - Enable "Maps SDK for Android"
#    - Create credentials → API Key
#    - Copy the API key → GOOGLE_MAPS_API_KEY
#    - Also add it to android/app/src/main/AndroidManifest.xml (see ENV_SETUP.md)
#
# 4. SMS PROVIDER SETUP (Supabase Phone Auth):
#    - Go to Supabase Dashboard → Authentication → Providers → Phone
#    - Enable Phone provider
#    - Configure Twilio or your SMS provider
#    - No need to add credentials here (configured in Supabase Dashboard)
#
# 5. After adding all keys:
#    - Save the .env file
#    - Restart Metro bundler: npm start -- --reset-cache
#    - Rebuild the app: npm run android or npm run ios
#
# IMPORTANT:
# - Never commit the .env file to version control
# - The .env file is already in .gitignore
# - Keep your keys secure
